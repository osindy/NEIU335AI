# CS 335: Artificial Intelligence (Spring 2026)

**Northeastern Illinois University**

## Course Overview
This course covers strong artificial intelligence methods resulting in systems that solve open problems in specialized domains. It explores AI models based on logical reasoning, probabilistic reasoning, and emergent intelligence, concluding with an in-depth look at language processing and modern AI applications.

## Major Topics
- **Foundations:** Search, Problem Solving, Logic, and Reasoning
- **Machine Learning:** Fundamentals, Naive Bayes, Decision Trees
- **Natural Language Processing (NLP):** Tokenization, Sentiment Analysis, semantic search
- **Modern AI:** Large Language Models (LLMs), Prompt Engineering, Retrieval-Augmented Generation (RAG)
- **Advanced Topics:** Reinforcement Learning, Evolutionary Computation, Swarm Intelligence
- **Ethics:** AI Safety and Real-World Considerations
